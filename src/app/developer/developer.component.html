Background
I started coding with ‘C++’ in my bachelors. At that time, it was compulsory subject and I looked at it as credit gaining course. After that I used very little action script while creating some small flash games in my college. At the starting 2012, I started coding in Perl and Matlab as I was doing project on Codons which required data mining and statistics. Later on, in 2015, shifted to Python for some serious computational biology projects. During the course of my PhD tenure I used to learn various programming languages whenever I get free time. This led to building up many programming projects, some of them are showcased here. Most of my projects are ‘Open Source’ and can be found on GitHub.
